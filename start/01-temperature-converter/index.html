Project Instructions

Please follow the instructions to build the Temperature Converter application.

Write JavaScript to achieve the following:

  • When a value is entered in the Celsius field, automatically convert it to Fahrenheit and display the result in the Fahrenheit field.
  • When a value is entered in the Fahrenheit field, automatically convert it to Celsius and display the result in the Celsius field.
  • Ensure that changing one input updates the other accurately without causing infinite loops or unintended updates.
  • You may change HTML if needed, but keep a note of what HTML did you change and why you needed to change.
